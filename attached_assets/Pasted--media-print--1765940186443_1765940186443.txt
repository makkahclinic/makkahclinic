الحل الأسرع (ويضمن صفحة واحدة): نقصّ الهوامش + نعمل “تصغير للطباعة” تلقائي داخل @media print + نقلل شوية المسافات وحجم الـ QR.

انسخ هذا البلوك وضعه داخل <style> عندك في آخره (أو استبدل قسم @media print بالكامل بهذا):

@page{
  size: A4;          /* لو تبغى A3: غيّرها A3 */
  margin: 6mm;       /* هوامش أقل عشان ما يكسر صفحة */
}

@media print{
  *{
    -webkit-print-color-adjust: exact !important;
    print-color-adjust: exact !important;
    animation: none !important;
    transition: none !important;
  }

  body{
    background: #fff !important;
    padding: 0 !important;
  }

  .print-btn{ display:none !important; }

  /* أهم نقطة: تصغير البوستر عشان يدخل صفحة واحدة */
  .poster-container{
    width: 100% !important;
    max-width: 100% !important;
    margin: 0 !important;
    border-radius: 0 !important;
    box-shadow: none !important;

    /* جرّب 0.90 ثم 0.88 لو لسه صفحتين */
    transform: scale(0.90) !important;
    transform-origin: top center !important;
  }

  /* تقليل المساحات */
  .poster-header{ padding: 22px 18px !important; }
  .poster-body{ padding: 18px 18px 14px !important; }
  .section-title{ margin-bottom: 10px !important; }
  .scan-badge{ margin: 8px auto 12px !important; padding: 10px 14px !important; }

  /* تثبيت شبكة 3 أعمدة وتقليل gap */
  .qr-grid{
    grid-template-columns: repeat(3, 1fr) !important;
    gap: 10px !important;
  }

  .qr-card{
    padding: 14px 10px 10px !important;
    box-shadow: none !important;
    break-inside: avoid !important;
    page-break-inside: avoid !important;
  }

  /* تصغير الـ QR */
  .qr-code-wrapper{ padding: 8px !important; }
  .qr-code-wrapper img{
    width: 120px !important;
    height: 120px !important;
  }

  /* تصغير النصوص شوي */
  .clinic-name{ font-size: 26px !important; }
  .section-title h2{ font-size: 22px !important; }
  .qr-title{ font-size: 18px !important; }
  .qr-desc{ font-size: 12px !important; margin-bottom: 10px !important; }
  .footer-text{ font-size: 12px !important; }
}


إذا بعد هذا لسه تطلع صفحتين: غيّر السطر:

transform: scale(0.90) !important;


إلى:

transform: scale(0.88) !important;
